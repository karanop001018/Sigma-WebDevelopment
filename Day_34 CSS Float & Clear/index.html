What is Web-Development

Web development refers to the process of creating, building, and maintaining websites. It includes aspects such as web design, web publishing, web programming, and database management1. It can be classified into two main categories:
Frontend Development: This is the part of a website where the user interacts directly, also referred to as the ‘client side’ of the application1. It involves the use of technologies like:
HTML: HyperText Markup Language, used to design the front end portion of web pages using markup language1.
CSS: Cascading Style Sheets, used to style the website1.
JavaScript: A scripting language used to provide dynamic behavior to the website1. Bootstrap: A free and open-source tool collection for creating responsive websites and web applications1.
Backend Development: This is the server side of a website, the part that users cannot see and interact with1. It is used to store and arrange data, and involves the use of technologies like:
PHP: A server-side scripting language designed specifically for web development1.
Java: A popular and widely used programming language that is highly scalable1.
Python: A programming language that lets you work quickly and integrate systems more efficiently1.
Node.js: An open source and cross-platform runtime environment for executing JavaScript code outside a browser1.
The process of web development also involves information gathering, planning, and design2. The goal is to create applications that work over the internet, i.e., websites
Frontend Development: Html, CSS and JavaScript
Backend Development: Python, SQL and APIs
Additional Skills: Problem Solving, DSA, Version Control and Testing/Debugging.
